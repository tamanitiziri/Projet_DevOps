\doxysubsection{Detailed Description}
Classe \doxylink{classSuffixArray}{Suffix\+Array}

Cette classe permet de construire un tableau de suffixes (SA) et un tableau de valeurs de préfixes les plus longs (LCP) pour une chaîne de caractères donnée. Elle permet également de rechercher un facteur dans la chaîne de caractères.
